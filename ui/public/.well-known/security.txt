# Scope
# -----
# Sprint EI-22 refresh — see https://aegisagent.in/security for the
# human-readable policy + rewards schedule.
#
# In scope:
#   * aegisagent.in + eu.aegisagent.in (production)
#   * staging.aegisagent.in (synthetic data only)
#   * any binary, container image, or SDK we publish (PyPI:
#     aegis-aevf, aegis-anthropic, aegis-openai, aegis-bedrock,
#     aegis-langchain)
#   * documentation that demonstrates a real attack
#
# Out of scope:
#   * customer tenant data (we cannot consent on their behalf)
#   * social engineering against staff or suppliers
#   * DoS, volumetric / network-layer testing without prior arrangement
#   * findings in third-party SaaS (Clerk, Stripe, AWS) — report
#     directly to that vendor
#
# Safe harbour
# ------------
# Reports made in good faith following this policy will not be
# prosecuted. We commit to:
#   * acknowledge within 48 hours
#   * triage within 5 business days
#   * publish a fix or mitigation within 90 days for High/Critical
#   * credit the reporter (with consent) in the next security advisory
