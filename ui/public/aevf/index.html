AEVF v0.1.0

Aegis Evidence Verification Format — an open spec for cryptographically signed audit bundles. Anyone with the public key can verify the bundle without contacting Aegis.

Verifier

The reference verifier ships as a pip package:

pip install aegis-aevf
aegis-verify --bundle reference-bundle-2026-06.json

Files served as text/plain / application/json. Indexed by your CDN. CORS open. No login required.
